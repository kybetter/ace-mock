## A fast local api-mock http service

#### feature
- file free
- easy to use
- quick to serve

#### install
```sh
$ yarn global add ace-mock
# or
$ npm i -g ace-mock
```

#### usage

```
$ ace-mock [options]
```

#### example
```sh
$ ace-mock
# or
$ ace-mock --port=8080
```
Then it will start a service to serve at:`http://localhost:12345`address.

Now, you can browse this page and set your api.
